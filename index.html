CSSでのレイアウトについて

横並びや、複雑なレイアウトを再現するには複数の方法があります。

  • tableレイアウト
  • inline-boxで横並び
  • floatとcreafix
  • FLEX BOX
  • GRIDレイアウト

今日はこの中の

  • tableレイアウト　レスポンシブしにくい
  • inline-boxで横並び　横並びのみ
  • floatとcreafix　なんかめんどくさいしイマドキじゃない
  • FLEX BOX
  • GRIDレイアウト

この二つを比較したいと思います。

表示速度

理論上は

GRID > FLEX BOX

GRIDは決まったレイアウトの中にコンテンツが読み込まれるのに対して、
FLEX BOXはコンテンツの大きさが決まった後にレイアウトが決まるから

検証では誤差程度しか差がでませんでした。

結論

GRID & FLEX BOX

どっちかだけを使うのではなく、臨機応変に適切な方を選ぶ

GRID = 2次元

FLEX BOX = 1次元

使い分け

オレンジ：FLEX BOX
青：GRID

よくある使い方

GRIDは複雑な配置ができる

GRIDは重なりを表現できる
